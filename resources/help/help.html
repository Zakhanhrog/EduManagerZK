EduZakhanh - Help & Information

Welcome

Welcome to the EduZakhanh Educational Management System! This guide provides information on navigating and using the application's features.

General Navigation

  • Use the tabs at the top (e.g., Schedule, Classes, Students) to access different management sections.
  • Data is primarily displayed in tables. Click on a column header to sort the data by that column. Click again to reverse the sort order.
  • Action buttons like Add, Edit, and Delete are typically located near the tables or data they affect.
  • Use the main menu bar (Main, Export, View, Help) for application-level actions like logging out, exporting data, changing themes, and accessing this help information.

User Roles and Permissions

The features and data you can access depend on your assigned role:

  • Administrator: Has full access to all sections, including managing user accounts and viewing system logs.
  • Teacher: Can view their schedule, manage students within their assigned classes, and view course information.
  • Student: Can view their personal schedule, enrolled classes, and general course information.

Core Features by Section

Administrator View

  • Schedule: View, add, edit, and delete class schedules. Use the date pickers and 'Load Schedule' button to view specific time ranges. Be mindful of potential conflicts (time/teacher/room).
  • Classes: Manage educational classes. Assign courses and teachers. Add or remove students from class rosters (enrollment). Set maximum class capacity.
  • Students: Manage student profiles (add, edit, delete). Ensure student phone numbers are correct, as they are used for account registration.
  • Teachers: Manage teacher profiles. Teacher IDs (provided during creation or editing) are needed for teacher account registration.
  • Courses: Define and manage the courses offered by the institution.
  • Rooms: Manage physical or virtual rooms, including capacity and type.
  • Accounts: View user accounts (Teachers, Students). Can reset passwords for users.
  • Logs: Review a chronological list of significant actions performed within the system (updates automatically).
  • Help: Access this help guide.

Teacher View

  • My Schedule: Displays your teaching schedule for the selected date range.
  • My Classes: Shows the classes you are assigned to teach and allows viewing enrolled students.
  • My Students: Provides a view of students enrolled in your classes.
  • Courses: Allows viewing details of available courses.
  • Help: Access this help guide.

Student View

  • My Schedule: Displays your personal class schedule.
  • My Classes: Shows the classes you are currently enrolled in.
  • Courses: Allows viewing details of available courses.
  • Help: Access this help guide.

Common Actions

  • Adding Data: Click the 'Add' button (or similar) in the relevant section. A dialog box will appear to enter the required information.
  • Editing Data: Select a row in the table you wish to edit, then click the 'Edit' button. A dialog box pre-filled with the current data will appear.
  • Deleting Data: Select a row in the table and click the 'Delete' button. You will usually be asked for confirmation. Note: Some items cannot be deleted if they are actively linked elsewhere (e.g., a teacher assigned to a class).
  • Filtering/Loading Data: Use date pickers or other filter options (if available) and click buttons like 'Load Schedule' or 'Filter' to narrow down the displayed data.

Other Features

  • Exporting Data: Administrators and Teachers can export data to Excel via the Export > Export to Excel... menu. Select the data type and choose a save location.
  • Changing Themes: Customize the application's look and feel using the View > Themes menu.
  • Logging Out: Use the Main > Logout menu item to securely log out of the application.
  • Exiting: Use the Main > Exit menu item or the window's close button to exit the application. You will be asked for confirmation.

Troubleshooting & Contact

If you encounter issues or have questions not covered here:

  • Ensure you have entered data correctly, especially required fields in dialog boxes.
  • Check the status bar at the bottom for any messages.
  • Contact the system administrator: zakhanh
  • Visit our online support portal.
