EduZakhanh - Help & Information

Welcome

Welcome to the EduZakhanh Educational Management System! This guide provides information on navigating and using the application's features.

General Navigation

  • Use the tabs at the top (e.g., My Schedule, Classes, Learning Results for Students; Schedule, Classes, Education for Teachers/Admins) to access different sections.
  • Data is primarily displayed in tables. Click on a column header to sort the data by that column. Click again to reverse the sort order.
  • Action buttons like Add, Edit, and Delete are typically located near the tables or data they affect, and are available based on your user role.
  • Use the main menu bar (Main, Export, View, Version) for application-level actions like logging out, exporting data, changing themes, and accessing this help information.

User Roles and Permissions

The features and data you can access depend on your assigned role:

  • Administrator: Has full access to all sections, including managing user accounts (Accounts), teacher profiles (Teachers), room information (Rooms), system logs (Logs), and all educational data.
  • Teacher: Can manage class schedules (Schedule), class information and student enrollment (Classes), student profiles (Students), course information (Courses), and has full access to manage grades, assignments, and view academic achievements for all students (Education tab). Teachers cannot access Accounts, Logs, Teacher management, or Room management.
  • Student: Can view their personal schedule (My Schedule), enrolled classes (My Classes), course information (Courses), and their learning results including grades and academic achievements (Learning Results tab).

Core Features by Section

Administrator View

Administrators have access to all features listed below, providing comprehensive control over the system.

  • Schedule: View, add, edit, and delete class schedules. Use the date pickers and 'Load Schedule' button to view specific time ranges. Be mindful of potential conflicts.
  • Classes: Manage educational classes, assign courses and teachers, manage student enrollment, and set class capacities.
  • Students: Manage student profiles (add, edit, delete).
  • Teachers: Manage teacher profiles (add, edit, delete).
  • Courses: Define and manage the courses offered.
  • Rooms: Manage physical or virtual rooms, including capacity and type.
  • Education:
    • Results & reviews: Select a class to view and manage student grades. Click Chỉnh Sửa (Edit) to modify scores, Art status, and Conduct. Save changes with Lưu Thay Đổi (Save Changes).
    • Academic Achievements: Select a class to view a list of students and their calculated academic achievements.
    • Assignments: Manage assignments for classes. Select a class, then Add, Edit, or Delete assignments.
  • Accounts: View user accounts (Teachers, Students) and reset passwords.
  • Logs: Review a chronological list of significant actions performed within the system.
  • Help: Access this help guide.

Teacher View

Teachers focus on academic management and have access to the following:

  • Schedule: View, add, edit, and delete class schedules for any class.
  • Classes: Manage educational classes, assign courses, and manage student enrollment for any class.
  • Students: Manage student profiles for students in any class.
  • Courses: View details of available courses.
  • Education: This is the primary section for teachers.
    • Results & reviews: Select any class from the tree to view and manage student grades. Click Chỉnh Sửa (Edit) to enable editing mode for scores, Art status, and Conduct. Click Lưu Thay Đổi (Save Changes) to commit them.
    • Academic Achievements: Select any class from the tree to view a list of students and their calculated academic achievements.
    • Assignments: Manage assignments for any class. Select a class from the dropdown, then Add, Edit, or Delete assignments. Assignments past their due date cannot be edited (only deleted).
  • Help: Access this help guide.

Note for Teachers: You do not have access to manage Teacher profiles, Room details, User Accounts, or System Logs. These are Administrator-level functions.

Student View

  • My Schedule: Displays your personal class schedule.
  • My Classes: Shows the classes you are currently enrolled in.
  • Courses: Allows viewing details of available courses.
  • Learning Results: This section provides your academic overview.
    • Info & Results: Displays your personal information and detailed academic transcript (grades for all subjects, average scores, conduct).
    • Academic Achievements: Shows your calculated academic achievements based on your performance.
    • Assignments: Lists assignments for your classes, including their due dates/times and current status (e.g., "Còn hạn làm bài", "Hết hạn làm bài").
  • Help: Access this help guide.

Common Actions (for Admin/Teacher)

  • Adding Data: Click the 'Add' button (or similar) in the relevant section (e.g., Students, Classes, Assignments). A dialog box will appear to enter the required information. Fields marked with * are typically mandatory.
  • Editing Data: Select a row in the table you wish to edit (e.g., a student, a class), then click the 'Edit' button. A dialog box pre-filled with the current data will appear.
    • In the Education > Results & reviews section, click Chỉnh Sửa (Edit) to make the grade table editable. Click on cells to input scores.
  • Saving Edits: After making changes (e.g., in the grade table or an edit dialog), click Lưu Thay Đổi (Save Changes) or "Save" to persist them. Use Hủy Bỏ (Cancel) to discard unsaved edits.
  • Deleting Data: Select an item and click the 'Delete' button. You will usually be asked for confirmation. Note: Some items cannot be deleted if they are actively linked elsewhere.

Academic Achievements Calculation

Academic achievements are determined based on the following rules (summary):

  1. Prerequisite: All core subjects (Toán, Văn, Anh, Lý, Hoá, Sinh, Sử, Địa, GDCD) must have a score of 2.00 or higher. If not, the title will be "Không đủ điều kiện xét".
  2. Part 1 (Overall Performance):
    • HS Yếu/Trung bình (Weak/Average Student): Overall average < 6.50; Conduct: Average or better.
    • HS Khá (Fairly Good Student): 6.50 ≤ Overall average < 8.00; Math, Literature, English ≥ 6.5; Conduct: Fair or better.
    • HS Giỏi (Good Student): 8.00 ≤ Overall average < 9.00; Math, Literature, English ≥ 8.00; Conduct: Good or Excellent.
    • HS Xuất sắc (Excellent Student): Overall average ≥ 9.00; Math, Literature, English ≥ 9.00; Conduct: Excellent.
    • If none of the above but prerequisite is met, "Đạt chương trình học tập" (Passed academic program) is shown.
  3. Part 2 (Specialization):
    • Bán chuyên KHTN (Semi-specialized in Natural Sciences): Natural Sciences average ≥ 9.00.
    • Chuyên KHTN (Specialized in Natural Sciences): Natural Sciences average ≥ 9.50.
    • Bán chuyên KHXH (Semi-specialized in Social Sciences): Social Sciences average ≥ 9.00.
    • Chuyên KHXH (Specialized in Social Sciences): Social Sciences average ≥ 9.50.

The final title combines Part 1 and Part 2 if applicable, e.g., Danh hiệu: "HS Giỏi" , "Bán chuyên KHTN". If only one part is achieved, only that part is shown. If neither part yields a specific title beyond "Đạt chương trình học tập", then "Không có danh hiệu nổi bật" may be displayed.

Other Features

  • Exporting Data: Administrators and Teachers can export data to Excel via the Export > Export to Excel... menu. Select the data type and choose a save location.
  • Changing Themes: Customize the application's look and feel using the View > Themes menu.
  • Logging Out: Use the Main > Logout menu item to securely log out of the application.
  • Exiting: Use the Main > Exit menu item or the window's close button to exit the application. You will be asked for confirmation.

  • Check the status bar at the bottom of the main window for any messages.
